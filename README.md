# Amazon-Website
A responsive Amazon-inspired e-commerce website built using HTML and CSS with custom UI design and interactive hover effects.
This project is a responsive Amazon-inspired e-commerce website built using HTML and CSS. It focuses on modern UI design and includes interactive hover effects to enhance user experience

Features
- Responsive homepage layout
- Navigation bar with search UI
- Product sections and cards
- Interactive hover effects for better UX
- Footer with multiple panels
- Clean and structured design

Tech Stack
- HTML
- CSS

Purpose
This project was developed to improve frontend development skills and understand real-world website layout and user interface design.

Future Improvements
Add JavaScript for dynamic features
Implement backend 
Add cart and authentication system
